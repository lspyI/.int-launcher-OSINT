# .int-launcher-OSINT
ОСИНТ лаунчер написанный на Python 3.7+
В нём собранны такие функции как: метаданные из файлов (ExifTool), поиск номеров (PhoneInfoga), геолокация по фото (GeoOSINT), поиск по username (Sherlock), поиск почт (Holehe), поиск по username (расширенный) (Maigret), метаданные из файлов v2 (EagleEye), настройки (Settings), выход (Exit).

У него открытый код и каждый новичок может посмотреть его и разобраться в нём из за пояснений.
Все функции не мои, это сборник функций для быстрой и удобной установки функций и использований их.
